OpenGL Stuff

Check out these cool screenshots from my raytracer, rollercoaster, and heightmap that I programmed for a computer graphics class using OpenGL. As of ACM SIGGRAPH 2023, I've been revisiting these projects and adding more stuff! Currently I'm working on adding tessellation to my heightmap!

Solo Project
C++, C, GLSL, OpenGL, 3D Math

Raytracer

  • Uniformly casted rays from camera location with one ray per pixel.
  • Wrote ray-object intersection code.
  • Implemented illumination equations.

Rollercoaster

  • Programmed camera move to follow the track of the rollercoaster.
  • Rendered track utilizing spline functions.
  • Created phong shader for track.
  • Created vertex and fragment shader to allow for texture mapping on the ground.

Height Map

  • Programmed vertex shaders to render smoothened and unsmoothened heightmaps.
  • Allowed user to change rendering modes from point, wireframe, and solid triangle.
  • Implemented 3D camera transformations, transformations of landscape (translate/rotate/scale).
